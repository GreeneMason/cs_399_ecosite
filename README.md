## ğŸš€ Installation & Setup

### Prerequisites

Before you begin, ensure you have the following installed on your system:

- **Node.js** (v18.0.0 or higher) - [Download here](https://nodejs.org/)
- **pnpm** (v10.0.0 or higher) - Install with `npm install -g pnpm`
- **Git** - [Download here](https://git-scm.com/)

### Step 1: Clone the Repository

```bash
git clone https://github.com/GreeneMason/cs_399_ecosite.git
cd cs_399_ecosite
```

### Step 2: Install Dependencies

```bash
pnpm install
```

This will install all required dependencies including Next.js, React, Tailwind CSS, and all UI components.

### Step 3: Run the Development Server

```bash
pnpm dev
```

The application will start on [http://localhost:3000](http://localhost:3000). Open this URL in your browser to view the site.

### Step 4: Build for Production (Optional)

To create an optimized production build:

```bash
pnpm build
```

To run the production build locally:

```bash
pnpm start
```
